---VocalCounter---

Este programa se encarga de realizar un contado de vocales a un archivo seleccionable.
Para la utilización de este programa se incluye un archivo (texto.txt) para utilizar como prueba.

---Lanzamiento---
Para inicar este programa se deberá llamar al módulo Principal.

Es necesario a la hora de lanzar el programa añadir como primer argumento la lozalización del módulo VocalCounter,
por ejemplo:

	java Principal C:\Users\usuario_x\Desktop\VocalCounter.java 

O si los módulos Principal y VocalCounter se encuentran en la misma carpeta:

	java Principal VocalCounter
